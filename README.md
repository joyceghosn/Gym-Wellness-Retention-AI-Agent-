# Gym-Wellness-Retention-AI-Agent-
Gym Wellness & Retention AI Agent
Project Overview

This project builds a full Gym Wellness & Retention AI System that combines:

Synthetic behavioural data generation

A machine learning model (logistic regression) that predicts the probability of a ‚Äúbad day tomorrow‚Äù

A Large Language Model (LLM) that generates personalized wellness explanations, 3-day micro-plans, and service recommendations

An automation workflow (n8n + Google Sheets + Twilio) that sends personalized SMS/WhatsApp retention messages

The goal is to:

Estimate member wellbeing risk using 7-day windows

Translate numerical risk into human-friendly coaching plans

Recommend one structured service offer from a fixed list

Automate retention messaging based on attendance, membership type, and birthdays

Provide gyms with a scalable AI blueprint for engagement, retention, and service promotion

Key Concepts

Behaviour is simulated using realistic rules

7-day rolling windows capture weekly patterns

Target variable: bad_day_tomorrow (binary)

Feature Engineering

The model uses the following engineered features:

mood_avg_7d

stress_avg_7d

fatigue_avg_7d

sleep_avg_7d

activity_days_7d

sleep_debt_7d

These features summarize an entire week of wellbeing indicators and are highly predictive of next-day difficulty.

Machine Learning Model

A logistic regression model predicts:

P(bad_day_tomorrow = 1 | last 7 days)

Risk Mapping

LOW: score < 0.4 ‚Üí performance strategy

MEDIUM: 0.4‚Äì0.7 ‚Üí balanced strategy

The ML output feeds directly into the LLM layer.

LLM Coaching System
1. Natural-Language Summary

Numerical features (e.g., sleep_avg_7d = 5.9) are converted to sentences like:

‚Äúsleep was short this week‚Äù

‚Äúfatigue is high‚Äù

‚Äúactivity was low‚Äù

2. Risk Explanation

GPT generates an empathetic explanation of why tomorrow may be difficult or easy.

3. Personalized 3-Day Micro-Plan

Includes:

sleep targets

specific activity recommendations

stress management techniques

4. Structured Offer Recommendation

Chosen from a fixed list:

yoga class

mobility class

pilates session

personal training session

nutrition consultation

stress-coaching session

This allows Power BI to track service popularity across risk segments.
